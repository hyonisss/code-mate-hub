# Code Mate v2.1.0

## ✨ Key Features

### 30% Improvement in Context Awareness
Significantly enhanced ability to understand the overall project structure. More accurate code suggestions are now possible.

### Multi-file Editing Support
Multiple files can now be modified simultaneously with a single request.

### Auto-generation of Korean Code Comments
A new feature that automatically adds Korean description comments to code.

## 🐛 Bug Fixes

- Fixed auto-completion delay in some environments
- Fixed Korean variable name recognition errors

## 📌 Known Issues

- Indexing may be slow for very large projects (1000+ files)
