# Code Mate v2.1.0

## ✨ 주요 기능

### 컨텍스트 인식 30% 개선
프로젝트 전체 구조를 이해하는 능력이 크게 향상되었습니다. 더 정확한 코드 제안이 가능합니다.

### 다중 파일 편집 지원
한 번의 요청으로 여러 파일을 동시에 수정할 수 있습니다.

### 한국어 주석 자동 생성
코드에 한국어 설명 주석을 자동으로 추가하는 기능이 추가되었습니다.

## 🐛 버그 수정

- 일부 환경에서 자동 완성이 지연되던 문제 수정
- 한글 변수명 인식 오류 수정

## 📌 알려진 이슈

- 매우 큰 프로젝트(파일 1000개 이상)에서 인덱싱이 느릴 수 있음
